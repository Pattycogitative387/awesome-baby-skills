# Contributing to Awesome Baby Skills 👶

## What We Accept

✅ AI agent skills that help with daily baby care routines
✅ Skills that track and learn individual baby patterns
✅ Skills that help organize information across caregivers
✅ Skills that curate age-appropriate content (books, activities, songs)
✅ Open source or free-tier with a public GitHub repo
✅ Compatible with at least one agent platform

## What We Don't Accept

❌ Medical diagnosis or treatment tools — even "screening" tools
❌ Skills that prescribe specific sleep training methods as "the right way"
❌ Baby product recommendation engines driven by affiliate revenue
❌ Generic parenting advice without personalization or memory
❌ Skills that create comparison anxiety ("your baby SHOULD be doing X by now")
❌ Anything without a working GitHub repository

## Safety Requirements

Every submitted skill MUST include a clear disclaimer that it does not provide medical advice. If a skill involves any health-related tracking (feeding, growth, symptoms), it must explicitly direct users to consult their pediatrician for any concerns.

## How to Submit

1. Fork this repository
2. Add your skill to the appropriate category in README.md
3. Use this format:

```
| [Skill Name](https://github.com/owner/repo) | What it does — be specific about what data goes in and what value comes out. | Platform1 · Platform2 |
```

4. Submit a PR with title: `Add [Skill Name]`
5. In your PR description, explain:
   - What baby data the skill works with
   - What it produces for the parent
   - How it handles the safety boundary (no medical advice)

## A Note on Sensitivity

New parents are exhausted, anxious, and doing their best. Every description, every PR comment, every piece of copy in this repo should be written with empathy. No judgment. No "you should be doing this." Just help.

The golden rule: make a tired parent's life easier, not harder.
